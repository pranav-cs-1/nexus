# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-12-07

### Added
- Initial release of Nexus TUI
- Full HTTP method support (GET, POST, PUT, PATCH, DELETE, HEAD, OPTIONS)
- Request organization with collections
- Response viewer with formatted JSON
- Complete request editing (URL, method, headers, query params, body, auth)
- Persistent storage using sled embedded database
- Keyboard-driven interface with Vim-like navigation
- Built-in example requests
- Export support (JSON collections and curl commands)
- Authentication support (Bearer, Basic, API Key)
- Welcome screen and help popup
- Clipboard integration for exports
